Описание задачи.Дан массив A размерности N. Необходимо:Найти индекс минимального элемента ($i_{min}$).Найти индекс максимального элемента ($i_{max}$).Определить границы интервала между ними.Вычислить сумму всех элементов $x$, таких что $x < 0$ и $x$ находится строго внутри этого интервала.   АлгоритмАлгоритм выполняется за два этапа:Поиск экстремумов: Линейное сканирование массива для нахождения позиций $min$ и $max$. В данной реализации используется поиск первого вхождения каждого из экстремумов.   Фильтрация и суммирование: Проход по подмассиву, ограниченному найденными индексами, с проверкой условия на отрицательность.   СложностьВременная сложность: $O(N)$, так как массив сканируется фиксированное количество раз (один проход для поиска индексов и один для суммирования).   Пространственная сложность: $O(1)$, так как мы используем лишь несколько переменных для хранения индексов и текущей суммы, не создавая копий массива.   Пример работыДля массива [30, -3, -7, 10, -2, 5, -10, 4]:Максимум: 30 (индекс 0).Минимум: -10 (индекс 6).Элементы между ними: [-3, -7, 10, -2, 5].Отрицательные элементы: -3, -7, -2.Результат: -12.ТребованияPython 3.x
